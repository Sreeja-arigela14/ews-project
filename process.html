V-FITTING

Procedure of making V-fitting

  1. The given mild steel flat piece is checked for given dimensions.
  2. One edge of given is filed with rough and smooth files and checked with a try square for straightness.
  3. An adjacent edge is also filed such that it is square to the first edge and checked with a try square.
  4. Wet chalk is applied on one side of the flat and dried for marking.
  5. Lines are marked according to the given figure, using an odd leg caliper and steel rule.
  6. Using the dot punch, punches are made along the marked lines.
  7. The excess material is removed from the remaining two edges with a try square level up to half of the man dots.
  8. Finally, burrs are removed by filing on the surface of the fitted job.

PRECAUTIONS:

  • The perpendicular of face ends edges is checked perfectly by using a try square.
  • Finishing is done using only smooth files.
  • Marking is done without parallax error.
